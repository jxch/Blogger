`MA5`代表今天`MA5`线的值
`MA5'`代表明天`MA5`线的值
`D'`代表明日收盘价
`D0`代表今日收盘价
`D1`代表昨日收盘价

`MA5 = (D0 + D1 + D2 + D3 + D4) / 5`
`MA5'= (D' + D0 + D1 + D2 + D3) / 5`

使 `MA5' > MA5` ，即明日`MA5`值大于今日`MA5`值（均线上行）
则 `(D' + D0 + D1 + D2 + D3) / 5 > (D0 + D1 + D2 + D3 + D4) / 5`
则 `D' > D4`

结论：若使明日`MA5`值大于今日`MA5`值，即`MA5`均线在明日上行，需满足明日收盘价大于4天前的收盘价的条件，那么`D4`就为`MA5`的抵扣价。
推论：若当天收盘价大于抵扣价，则均线上行
应用：在一个时间窗内，推演均线运行轨迹
